Why this data¶

Using the best neural network player that our Alpha Awari was able to produce, three datasets have been created in order to better analyze the strenght and understand how many times the network is able to choose the correct move. In this work an analysis on the three datasets is provided.

The three datasets are:

  • Matches dataset, dataset that describe how many optimal and suboptimal moves the network can choose on each match.
  • Moves dataset, dataset that describe in deep the moves of our work.
  • Opponent dataset, summarize the number of wins or lose for each different opponent.

Dataset Description¶

Matches dataset¶

In the following table the meaning of each column is explained:

Column Name

Column Description

Match

Progressive number of matches.

Optimal_moves

Number of optimal moves selected in the match.

Sub_optimal_moves

Number of sub-optimal moves selected in the match.

Win

1 if the neural network win the match 0 otherwise.

Number_of_turns

Total number of turns in the match.

Moves dataset¶

In the following table the meaning of each column is explained:

Column Name

Column Description

Board

Represent the number of seeds of each pit.

best_move_selected

1 when the best move is selected 0 otherwise.

Loss

Points lost with this move.

Move_selected

Move selected from the neural network.

p-value

Policy values based on the neural network evaluation.

Opponent dataset¶

In the following table the meaning of each column is explained:

Column Name

Column Description

Name of the opponent.

Win_p1

Number of wins when the neural network plays first.

Lost_p1

Number of match lose when the neural network plays first.

Draw_p1

Number of match draw when the neural network plays first.

Win_p2

Number of wins when the neural network plays second.

Lost_p2

Number of match lose when the neural network plays second.

Draw_p2

Number of match draw when the neural network plays second.

Optimal_moves_nn

Number of optmial moves selected with the opponent.

Sub_optimal_moves_nn

Number of sub-optmial moves selected with the opponent.

def annotate_plot(plot, x=.18, y=.2, font=15):
    # create a list to collect the plt.patches data
    totals = []

    # find the values and append to list
    for i in plot.patches:
        totals.append(i.get_height())

    # set individual bar lables using above list
    total = sum(totals)

    # set individual bar lables using above list
    for i in plot.patches:
        # get_x pulls left or right; get_height pushes up or down
        plot.text(i.get_x()+x, i.get_height()+y, str(round((i.get_height()/total)*100, 2))+'%', fontsize=font, color='dimgrey')
